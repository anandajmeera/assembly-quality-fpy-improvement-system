# Assembly Line Quality Monitoring & First Pass Yield Improvement System | Tableau
This project presents a Tableau-based analytics dashboard that analyzes manufacturing defects recorded between September 2025 and November 2025. It focuses on defect types, severity, repair costs, monthly trends, and the effectiveness of inspection methods to support data-driven quality improvement.
